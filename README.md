# SimpleScan
基于zxing的扫一扫功能，可在一定程度上实现自定义界面

#### 一、说明：

该项目基于google的zxing扫码库，在此基础上实现了扫描界面的简单自定义，扫码框大小可控，网络监测，开关闪光灯功能等；


#### 二、使用方法：

将项目中的libzxing作为module导入到自己的项目中，CaptureActivity为扫描界面，进行自定义界面也是在该界面中完成。
